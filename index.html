Star Fan

Proyecto desarrollado por un equipo de 4 integrantes. El trabajo que se realizó como FrontEnd fue el de utilizar una API pública de películas y crear un producto, el tema era libre. Trabajé con la API de Youtube, para obtener videos de canales afines a la temática de Star Wars. El proyecto se desarrolló en HTML, utilizando JavaScript, Jquery, el framwork de Bootstrap, la plataforma de Firebase y la aplicación de API públicas de OMDb API y la API de Youtube.

Easy-VReco

Proyecto en dónde se debía reproducir un mapa utilizando la geolocalización e inputs con autocompletado dónde el usuario al poner datos de origen y destino se pudiese trazar una ruta en el mapa y también la aplicación puede mostrar dónde el usuario se encuentra. Para el desarrollo de este trabajo utilicé la API de geolocalización y la API de Google Maps, el framework de Materialize, la librería de JQuery y con el lenguaje JavaScript.
